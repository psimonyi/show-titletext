Ways to add rules

Right-click on an image and choose 'Set title-text rule with this image'. A rule for the current domain will be added (or replaced) with a selector that would find that image.
When there is no rule set for the current domain but the main image on the page has title-text, click the icon in the location bar to add a rule for it. Enabling this option may slow down page loading slightly.
